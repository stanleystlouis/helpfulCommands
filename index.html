Git Git is the most commonly used version control system. Git tracks the changes you make to files, so you have a record of what has been done, and you can revert to specific versions should you ever need to. Git also makes collaboration easier, allowing changes by multiple people to all be merged into one source

Linux Terminal The use of the Command Line Interface (CLI) commonly known as terminal becomes more apparent as one gains experience in programming. Here are some benefits of using the terminal as opposed to Graphical User Interface (GUI). Remote connections become easier, one can connect using ssh through the terminal via OpenSSH. Open-source applications usually first support the terminal before GUI. Linux servers only support the terminal.
