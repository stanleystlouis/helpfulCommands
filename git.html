Git Cheatsheet

Git Basics:
mkdir <directory-name> Make directory, folder
cd <directory-name> Change directory
git init Initialize directory
ls list contents of file or directory
touch <filename> Create file
git status get files status
git add <filename> add file to staging area
git add . add all files to staging area

Commit:
git commit commit file with editor
git commit -m <description> commit from the command line
git commit --amend amending a commit
git log view log of all commits
git log --oneline view in one line
Branch:
git branch Displays branch
git branch -r commit from the command line
git branch <branch-Name> create new branch
git switch -c <branch-name> create and enter branch
switch between branches switch between branches
Pull:
git pull <remote> get latest changes to local repo without integration with existing code
git pull <remote> <branch-name> get latest changes to local repo without integration with existing code
Push:
git push origin <branch-name>
git push origin <local>:<destination>
git push -u origin
Restore:
git restore <filename> delete uncommit changes in file
git restore --staged <filename> unstage file
git restore --source HEAD~<commit #> <filename> restore up to that commit
Useful Github Funtionalities:
Git gist Host snipet of code
Host pages host pages on Github
Pull request Request merge access
Forking copy a repo to your directory

Please visit Git Docs for more details
